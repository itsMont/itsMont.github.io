Inducción Matemática

La inducción matemática determina que algo se cumple para una base (un elemento inicial) y del mismo modo para un elemento arbitrario y el siguiente elemento

Por ejemplo:

  1. Universo: N (Naturales)
    Una propiedad se cumple para todos los números naturales si y sólo si:
  • Se cumple para 0 (Base)
  • Si se cumple para un nnnn arbitrario, entonces se cumple para n+1n+1n+1n+1

También puede verse como
2. Universo: N (Naturales)

  • Se cumple para 0 (Base)
  • Si se cumple para los menores que un nnnn arbitrario, entonces se cumple para nnnn

Entonces en cierto caso se puede llegar a lo siguiente.
Ejemplo de propagación (Un paciente n puede contagiar al paciente n+1)

  • Se cumple para 0
  • Si se cumple para los menores que 1, se cumple para 1
  • Se cumple para 1 (Porque se cumple para 0 y 0 < 1)
  • Si se cumple para los menores que 2, se cumple para 2
  • Se cumple para 2
  • Si se cumple para los menores que 3, se cumple para 3
  • Se cumple para 3
  • ...

Relaciones de equivalencia

Una relación es de equivalencia si es:

  • Reflexiva
    • Para todo xxxx, xxxx se relaciona consigo mismo: xRxxRxxRxxRx
  • Simétrica
    • Se tiene cuando xRyxRyxRyxRy al mismo tiempo que yRxyRxyRxyRx
  • Transitiva
    • Si xRyxRyxRyxRy y yRzyRzyRzyRz entonces xRzxRzxRzxRz
